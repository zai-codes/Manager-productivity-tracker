Features
- Account Management: Email/Google sign‑in, secure login/logout
- Task Management: Create, read, update, delete (CRUD) tasks
- Task Prioritization: Categories for high/medium/low priority
- Deadlines & Reminders: Notifications for upcoming tasks
- Progress Tracking: Charts/graphs for completed tasks & time spent
- Daily Productivity Notifications
- Offline Usage with auto‑sync when online
- Cloud Backup & Multi‑Device Sync
- Export Options: PDF/CSV reports
- UI Options: Dark mode, clean minimal design
Tech Stack
- Language: Kotlin
- Architecture: MVVM (Model‑View‑ViewModel)
- Database: Firebase Firestore + Room (local)
- UI: Material Design Components
- Other Tools: Firebase Authentication, WorkManager for reminders

 Future Enhancements
- AI‑powered productivity insights
- Collaboration features (shared tasks)
- Integration with calendars (Google Calendar, Outlook)
